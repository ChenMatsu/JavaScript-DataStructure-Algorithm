# Hash Table

> Used to store key-value pairs

-   Feature
    -   Keys are not ordered

---

-   Languages
    -   Python: Dictionary
    -   JS: Objects and Maps
    -   Java, Go & Scala: Maps
    -   Ruby: Hashes

---

-   Theorem
    -   Hash Function
    -   Handling Collision: Collisions are inevitable
        -   Seperate Chaining
        -   Linear Probing

---

-   Time Complexity (Depends on how good Hash function is)
    -   Insert: O(1)
    -   Deletion: O(1)
    -   Access: O(1)
