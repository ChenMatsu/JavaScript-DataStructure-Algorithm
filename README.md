# Stack and Queue

## Stack
> LIFO (Last-In-First-Out)
- Manage Function Invocation
- Undo / Redo
- Browser History Routing
- Time Complexify
  - Insertion: O(1)
  - Removal: O(1)
  - Search: O(N)
  - Access: O(N)

## Queue
> FIFO (First-In-First-Out)
- Background Tasks
- Uploading Resources
- Printing / Task Processing
- Time Complexify
  - Insertion: O(1)
  - Removal: O(1)
  - Search: O(N)
  - Access: O(N)
